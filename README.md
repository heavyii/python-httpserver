# http server for python

python3 简单 http服务器

## 技术栈

- [python的http服务器](https://bottlepy.org/docs/dev/)
- [apidoc](https://apidocjs.com/#getting-started)

## 启动服务器

    pip install bottle
    python main.py

* api文档 http://localhost:8080/static/index.html

## 编译api文档

    npm instal
    npm run doc
